# CHALLENGE API

## INSTRUCTIONS

### LOCAL

1. Execute "npm install"
2. Configure "src/app.config"
3. Execute "nodemon src/apichallenge.ts"

### DEPLOY

1. Execute "npm install"
2. Configure "src/app.config"
3. Execute "npm run build"
4. Execute "npm install -g pm2"
5. Execute "pm2 start build/apichallenge.ts"
